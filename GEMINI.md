# Project Overview

This project is a web application built with Go and several modern web technologies.

## Technologies Used

*   **Backend:** Go with the Gin framework
*   **Frontend:** HTMX for dynamic content loading
*   **CSS:** PicoCSS for styling
*   **Data:** `data.json` for storing portfolio data

## File Structure

```
/
├── data.json
├── go.mod
├── go.sum
├── main.go
├── static/
└── templates/
    └── index.html
```
