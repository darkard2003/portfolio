# Portfolio Website

This is a personal portfolio website built with Go and a modern frontend stack. It showcases my projects, skills, and contact information.

## ✨ Features

-   **Dynamic Content:** Built with Go and the Gin framework for a fast and robust backend.
-   **Modern Frontend:** Uses HTMX for dynamic content loading, providing a smooth user experience without full page reloads.
-   **Minimalist Design:** Styled with PicoCSS for a clean and responsive layout.
-   **Data-Driven:** Portfolio data is managed in a simple `data.json` file, making it easy to update.

## 🛠️ Technologies Used

-   **Backend:** Go, Gin
-   **Frontend:** HTMX, JavaScript, PicoCSS
-   **Data:** JSON
